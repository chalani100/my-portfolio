# Personal Portfolio Website

A lightweight React + Vite + Tailwind CSS v4 personal portfolio website for **Lakeesha Chalani Gamage** (Index: **23IT0462**).

## How to Run the Project Locally

If you received this project as a ZIP archive, follow these steps to run it on your local machine:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (Node 18+ is recommended).

### Steps:

1. **Extract the ZIP file** to a directory of your choice.
2. **Open your Terminal** (Command Prompt, PowerShell, or Bash) and navigate to the extracted folder:
   ```bash
   cd path/to/portfolio
   ```
3. **Install the dependencies**:
   ```bash
   npm install
   ```
4. **Run the local development server**:
   ```bash
   npm run dev
   ```
5. **Open the site**:
   Once the server starts, open your browser and go to the local address shown in your terminal (usually `http://localhost:5173/` or `http://localhost:5174/`).

## Project Structure
- `src/App.jsx` - The main single-file React component containing all layout structures and logic.
- `src/index.css` - Stylesheet importing Tailwind CSS v4 and enabling smooth scroll behavior.
- `public/profile.jpg` - Profile photo displayed in the Hero section.
